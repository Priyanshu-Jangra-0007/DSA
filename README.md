# Data Structures & Algorithms (DSA)

This repository contains my practice and implementation of **Data Structures and Algorithms** using **Java**, with a strong focus on writing clean, efficient, and optimized solutions.

The primary objective of this repository is to strengthen problem-solving skills and prepare for technical interviews by mastering core algorithmic patterns.

---

## Focus Areas

- Dynamic Programming (recursion, memoization, tabulation)
- Two Pointer techniques
- Recursion and Backtracking
- Sliding Window problems
- Binary Search optimizations
- Greedy algorithms

Each problem is solved with an emphasis on:
- Correctness
- Time and space complexity
- Readability and reusability
